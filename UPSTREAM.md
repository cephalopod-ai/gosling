# Upstream Provenance

gosling is an independent fork of [goose](https://github.com/aaif-goose/goose), the open source AI agent project originally created by the Agentic AI Foundation (AAIF).

This repository is not affiliated with, endorsed by, supported by, or published by the goose project or AAIF. Current ownership, support, releases, package metadata, and documentation should point at the gosling fork unless a reference is explicitly preserving upstream credit, compatibility history, or license provenance.

## Imported Source

- Upstream project: goose
- Upstream repository: https://github.com/aaif-goose/goose
- Imported baseline: goose v1.38 as documented in this repository's README
- License: Apache License 2.0

## Preservation Policy

- Preserve upstream copyright notices, license text, attribution, and historical references.
- Preserve explicit compatibility notes for inherited formats such as old share links when the product still supports them.
- Do not route current support, security reports, release downloads, telemetry, package publishing, or maintainer review to upstream goose or AAIF unless a future maintainer intentionally re-establishes that relationship and documents it here.
